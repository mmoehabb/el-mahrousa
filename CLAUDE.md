# CLAUDE.md

## Design Context

### Users

Egyptian board game enthusiasts playing with friends and family. Players range from casual gamers to competitive friends who enjoy strategic trading and property building. The game is played in social settings - either local (pass-and-play) or online P2P.

### Brand Personality

**Retro Pixel Art** - Nostalgic, playful, and authentic. The interface channels classic 8-bit gaming aesthetics while celebrating Egyptian culture. The pixel art style evokes nostalgia for retro games while remaining fresh and distinctive.

### Aesthetic Direction

- **Visual Tone**: Retro gaming meets Egyptian heritage. Pixel fonts (Press Start 2P, Lalezar) combined with pharaonic color palette and patterns
- **Theme**: Both light and dark modes supported, defaulting to system preference. Light mode uses warm sand tones (#f4a460) with papyrus texture; dark mode uses deep blue with gold accents
- **References**: Classic 8-bit board game adaptations, retro gaming UIs
- **Anti-references**: Generic modern SaaS aesthetics, corporate banking apps, overly minimalist designs

### Design Principles

1. **Embrace the Retro** - Pixel fonts are central to the identity. Use them for headings and game elements; ensure readability at smaller sizes
2. **Cultural Authenticity** - Egyptian theming (gold, blue, red, pharaonic patterns) should be visible but not overwhelming. The game feels Egyptian, not like a museum
3. **Enhanced Accessibility** - Larger touch targets (min 44px), higher contrast ratios, clear visual hierarchy. Support reduced motion preferences
4. **Warm & Inviting** - The interface should feel like gathering around a board with friends. Use the papyrus texture and gold accents to create warmth
5. **Clarity Over Complexity** - While the retro aesthetic is bold, game information must be instantly readable. Use contrast and spacing to prioritize readability
